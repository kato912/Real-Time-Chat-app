# Real time Chat app
### แอพแชทแบบreal time
------------------------
#### Framework ที่ใช้งาน
##### 1. pyqt6 GUI framework
```
pip install pyqt6
```
##### 2. Flask backend Framework
```
pip install flask
```
----------------
#### การใช้งาน
ต้องรันไฟล์ server.py ก่อนจึงเริ่มจึงเริ่มรันไฟล์ client.py
